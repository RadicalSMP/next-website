# Next Website

这是 **根号的离谱服务器**网站 的仓库。

## 技术栈

### 前端

- **Bun** 包管理器 & Js运行时
- **Next.js**
- **Typescript**
- **TailwindCSS**
- **shadcn/ui**

### 后端

- **Next.js** API 路由
- **uv** 包管理器 & 虚拟环境管理
- **PostgreSQL**

### 托管

- **Vercel** 托管 Next.js + Fastapi 项目
- **supabase** 托管 PostgreSQL 数据库

## 本地运行

### 安装依赖

> 确保你已经安装了 bun

```bash
bun install
```

### 运行项目

```bash
bun run dev
```
